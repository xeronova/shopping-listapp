# 🛒 쇼핑 리스트 앱

아이템 추가, 삭제, 체크 기능이 있는 간단한 쇼핑 리스트 웹 앱입니다.

## 기능

- **아이템 추가**: 텍스트 입력 후 `+` 버튼 클릭 또는 `Enter`
- **체크/언체크**: 체크박스 클릭으로 완료 처리 (취소선 + 흐리게)
- **개별 삭제**: 각 항목의 `✕` 버튼
- **완료 항목 일괄 삭제**: "완료 항목 삭제" 버튼
- **영속성**: `localStorage`로 새로고침 후에도 목록 유지

## 실행 방법

`index.html` 파일을 브라우저에서 직접 열면 됩니다. 별도 서버나 의존성 없음.
